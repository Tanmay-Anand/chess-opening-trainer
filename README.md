#Chess Opening Trainer:

Clean, interactive web app that helps beginners like me to learn classical chess openings move-by-move.  
Built with HTML, CSS, and JavaScript WITH no libraries, no frameworks, just a polished UI and smooth logic.

#Live Demo  
[Netlify Link]((https://chessopeningprep.netlify.app/))

#Features
- Move-by-move guidance for each opening.
- Highlights correct moves.
- Error messages for wrong attempts.
- "Show Hint" button for beginners.
